Each e-model detailed page page has a section for the ion channels that are present in the model. Go to Data, select Public then select Model and E-model. Select the e-model for which you want to find the ion channel models. that you would like to explore and click View details. Once, on the detailed E-model page, click on the Configuration tab on the left and scroll down to the bottom to see the ion channel models used in the e-model. 
If you explore the ion channel model used in me-model, Go to Data > Seelct Public > Model > Select the ME-model > View Details > Configuration. You will see the E-model used. Click on the More details link on the E-model tab. This will open the detailed page of the E-model. Here, go to Configuration and scroll down to see the Ion Channel models.
