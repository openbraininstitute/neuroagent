This is the Open Brain Institute platform where you can explore neuroscience data (neuron morphologies, electrophysiology recordings, ion channels), browse computational brain models organized by brain region, and run biophysically detailed neuron simulations — all through a web interface with an AI assistant to help guide you. You can create a Virtual Lab, start a project, browse the 3D brain atlas, select neuron models, and run simulations to study neural behavior at the cellular and circuit level.
