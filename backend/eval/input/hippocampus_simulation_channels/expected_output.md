You can start such simualtions by blocking ion channel by setting the conductance of channel to zero. This can be done in Single Neuron (beta) Workflow. Go to Workflows > Simulate > Single Neuron (beta). In the model selection page look for a hipppocampal pyramidal neuron. At present, the public rat and mouse hippocampus ME-Models have the brain region `Field CA1`, e-type `cACpyr` (for rat models and `cADpyr` for mouse models) and m-type `SP_PC`. Select one such pyramidal neuron model. After setting the `Stimuli`, select modifications. Under modification, you can select the level of modification: section list, section or compartment and then select the respective options e.g. apical or basal (section list), apic[10] (section) and dend[2](0.3) (compartment) level. Once the options are selected, you can select the variable to modify e.g. `gbar_nax` and set the value to 0. This variable is the maximum conductance of the nax channel and this a zero value is similar to blocking the nax channel. You can also do a parameter scan over conductance variable to simulate partial blocks or other variables such stimulus current. Then can `Generate Simualtion` and `Launch Simulation` to start the simualtion. The results for different simualtions will appear in the output tab.
