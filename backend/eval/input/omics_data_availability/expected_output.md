The Open Brain Platform is centred on digitally reconstructing and studying brain tissue with biophysical modelling: morphologies, patch-clamp‑style electrophysiology, neuron and circuit models, simulation workflows and related experimental annotations. Multi‑omics in the broad sense—large transcriptomic, proteomic or spatial‑omics atlases treated as standalone omics warehouses—is not the platform’s primary product lane compared with those modelling assets.

You can still browse **Explore → Model data** (and dataset or project listings) for specific holdings that mention molecular/transcript‑level modalities, run literature search where appropriate, and read the latest product docs via the assistant. Dedicated patch‑seq holdings are sparse to absent today; broader “omics atlas” hunting should rely on Explore search plus external archives rather than expecting a turnkey omics catalogue here.
