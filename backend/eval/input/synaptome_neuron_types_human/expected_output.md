The synaptomes data is present under `Data > Public > (root brain region) > Model > Synaptome` and `Synaptome (beta)`. The ones under Synaptome are artificially built synaptomes. These use single neuron models (ME-Models) from different brain region including straitum with different synapse sets. You can build your own using Workflow>  Build > Synaptome. The ME-model used can be from cigulate cortex and substantia nigra which will differntiate the synaptome. You can check the configuration of a synaptome by going to the detailed page to find the Me-Model used and synapse sets added. The paltform currently only has mouse striatal ME-models and Syanptomes.
